Now there are many clubs that have formed for different kinds of activities and there are also many variations of clubs that run across the world. And the management system is vastly used for managing their activities. But there is no management system that is used by a club. So then we have taken the step to make a system for all types of clubs. So I have developed a Web application for The Study Buddies Club Membership System.

Purpose of this app

In this The Study Buddies Club Membership System, the website provides a sign up page which if the user has not been a member for the club, the user can register themselves at the website. The interesting part is, the system is not only a club membership system. So the main goal is to make a club fully technology based that can be easily handled. So that many of the clubs can maintain their activity easily and smoothly. On the other hand though, it is an open source project so that they can rearrange all the components according to their needs.

Development

This system is build using Php and MYSQL as the database.
