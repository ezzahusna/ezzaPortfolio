Penang Travel Service Provider Application is a tourism application that is developed to ease the work of travellers or backpackers. This mobile application is integrated with geofencing technology that enables users or travellers to detect attractions around them in specific areas

Purpose of this app

Alongside with the technology integrated, this mobile application has a few functionalities such as enabling users to book a hotel, make vehicle reservation, buying tickets through a directed link to its own website also consist of an information booth inside the application which makes it a multifunctional mobile app.

Development

This application was developed based on Android platform as the operational system which requires Android developing software which is Android Studio. Furthermore, Firebase also was used for cloud database purposes. The development of this mobile application has been done using Agile as its methodology
