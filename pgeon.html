Managing couriers using traditional techniques, such as recording client data, will use a lot of human resources and need the consumer to fill out a form manually. This will produce or establish a courier office with a large number of individuals waiting to pick up an item. As a result, I have developed the Pgeon Courier Management System as a solution to create a system. It can provide several advantages to both the consumer and the firm.

Purpose of this app

the system creates data such as supplier and employee information. Furthermore, the system includes two types of forms: one for entering a new record into the system and another for updating the data included in the record. The system can also do certain CRUD actions (Create, Read, Update, Delete) to make it easier for individuals to maintain their records.

Development

This system is build using java and MVC programming pattern.
